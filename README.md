# 路上 - 房车生存文字游戏

一个纯 HTML/CSS/JS 实现的房车生存文字游戏，无需后端，浏览器直接运行。

## 游戏简介

你是一个驾驶房车旅行的人，在公路上寻找生存的意义。管理资源、应对天气、探索地点、做出选择，体验一段独特的旅程。

## 核心系统

- **生存数值**：饱食、饥饿、温暖、精力、心情
- **资源管理**：燃油、水箱、电池、金钱
- **食材食谱**：9个专属食谱（野外采集食材，营地烹饪）
- **水箱系统**：烹饪/洗澡消耗用水
- **房车改装**：9部位改装，属性加成+效果触发
- **房车升级**：5 级房车，解锁更多功能
- **储能系统**：太阳能充电+电池日消耗
- **随机事件**：89 个事件（驾驶/休息/天气/节日/地点关联/闲置）
- **存档系统**：localStorage 自动保存

## 运行方式

直接用浏览器打开 on the road.html 即可开始游戏。

## 技术栈

- 纯 HTML + CSS + JavaScript
- 无框架，无依赖
- 单文件实现全部逻辑
